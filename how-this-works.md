# How this all works — plain English

## The folder
Your website lives in this folder on your computer. Two HTML files and some images. That's it.

## GitHub
A backup system with memory. Every change you save to it is remembered forever, so nothing gets lost.

**GitHub Desktop** is the app that syncs your folder to that backup. You only need to open it after changes are made.

## VS Code
A text editor for HTML files. You don't need to use it — Claude handles that.

---

## How to work on your site with Cowork

1. **Open the Cowork chat** (Claude)
2. **Tell Claude what you want** in plain English — "add my story about the candidate who…", "change the wording on chapter 3", "fix the email button"
3. **Claude edits the files directly** in your folder
4. **Open GitHub Desktop** — it will show the changes automatically
5. **Press "Commit to main"** (add any note you like, or just leave the default)
6. **Press "Push origin"**

That's it. Two buttons. Your site is backed up and up to date.

---

## What Claude can do for you

- Write and edit the guide content based on your stories
- Change wording, layout, colours
- Add images
- Fix broken buttons or links
- Build new pages

You just describe what you want. No code required.
